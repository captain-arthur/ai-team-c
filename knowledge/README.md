# Knowledge

## 개념

Knowledge는 AI 팀이 축적하는 재사용 가능한 지식입니다.

프로젝트 수행 과정에서 학습한 내용을 체계적으로 기록하여 미래 작업에 활용합니다.

## 지식 유형

### design-principles

설계 원칙과 철학

- 아키텍처 원칙
- 코딩 원칙
- 인터페이스 설계 원칙

### patterns

반복적으로 활용되는 해결 패턴

- 설계 패턴
- 구현 패턴
- 운영 패턴

### lessons-learned

경험에서 배운 교훈

- 성공 사례 분석
- 실패 사례 분석
- 의사결정 회고

### domain-knowledge

특정 도메인에 대한 전문 지식

- 기술 도메인
- 비즈니스 도메인
- 운영 도메인

### references

참조 자료 및 외부 지식

- 유용한 문서 링크
- 도구 사용법
- API 레퍼런스

## 지식 문서 템플릿

```markdown
# {제목}

## 요약
핵심 내용 한두 문장

## 상세
자세한 설명

## 적용 상황
이 지식이 유용한 상황

## 관련 항목
- 관련 지식 링크
- 관련 프로젝트 링크

## 출처
학습한 경로나 참조
```

## 지식 관리 원칙

### 기록 시점

- 새로운 통찰을 얻었을 때
- 문제를 해결한 후
- 실패에서 교훈을 얻었을 때
- 유용한 패턴을 발견했을 때

### 작성 기준

- 재사용 가능하도록 일반화
- 구체적인 예시 포함
- 적용 맥락 명시
- 간결하고 명확하게

### 활용

- 새 프로젝트 시작 전 관련 지식 확인
- 의사결정 시 과거 교훈 참조
- 정기적으로 지식 정리 및 업데이트
